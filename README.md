# Bolsa de Empleo CIAF

Plataforma estudiantil de empleo para CIAF Pereira.

## Estructura del proyecto

```
bolsa-empleo-ciaf/
├── index.html
├── vercel.json
├── vite.config.js
├── package.json
└── src/
    ├── main.jsx
    └── App.jsx
```

## Cómo correr localmente

```bash
npm install
npm run dev
```

## Cómo subir a Vercel

### Opción A — Vercel CLI (recomendado)
```bash
npm install -g vercel
vercel
```

### Opción B — Vercel Dashboard
1. Sube esta carpeta a un repositorio de GitHub
2. Entra a https://vercel.com/new
3. Importa el repositorio
4. Vercel detecta Vite automáticamente → haz clic en **Deploy**

¡Listo! Tu sitio estará en línea en menos de 2 minutos.
