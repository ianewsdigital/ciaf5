# Bolsa de Empleo CIAF

Plataforma estudiantil de empleo para el Centro Integral de Formación — Pereira, Risaralda.

## Stack

- React 18
- Vite 5
- Vercel (deploy)

## Correr localmente

```bash
npm install
npm run dev
```

## Subir a Vercel

### Opción A — Vercel CLI
```bash
npm install -g vercel
vercel
```

### Opción B — GitHub + Vercel (recomendado)
1. Sube esta carpeta a un repositorio de GitHub.
2. Ve a [vercel.com](https://vercel.com) → **Add New Project**.
3. Conecta el repositorio.
4. Vercel detecta Vite automáticamente. Haz clic en **Deploy**.

¡Listo! Vercel te da una URL pública en segundos.
